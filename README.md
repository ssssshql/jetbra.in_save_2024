## Development

This project uses [Vue 3](https://vuejs.org/) + [Vite](https://vitejs.dev/).

### Prerequisites

*   Node.js (v18 or higher recommended)
*   pnpm

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm run dev
```

### Build

```bash
pnpm run build
```
